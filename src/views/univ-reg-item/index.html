University Scheduler A University Registration System

Project Description

The University Scheduler is a registration and enrollment system with functionality for multiple user login, creating student/faculty directories, course catalogs, and output class schedules. This was a large Java project, implemented over the course of the semester and covered several concepts of object-oriented programming. View the source code on Github
