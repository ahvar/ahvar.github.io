The Help Desk Tracking Technical Support Requests

Project Description

The help desk is a tool used by the IT staff to support the technical needs of a company's employees. The system uses a finite state machine design to represent the different states and transitions that occur during the technical support process. A user can add new tickets, assign work requests, make progress notes, collect feedback, and complete other support related tasks. This was the second project in CSC 216 - Programming Concepts, at NCSU. See the ReadMe.md and source code on Click Github
