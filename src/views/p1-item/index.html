Ecosystem Simulation How an ecosystem evolves over time

Project Description

Object-oriented languages empower developers and analysts with the abiilty to model real-world processes and observe how those systems change over time. This project, the first from CSC 216 - Programming Concepts, required development of an ecosystem where different types of animal populations in a food chain could be programmed with basic behaviors and set in motion for a specific amount of time. An understanding of fundamental concepts such as polymorphism, inheritance, and composition were important for the implementation of the ecosystem. You can view the source code on Github
